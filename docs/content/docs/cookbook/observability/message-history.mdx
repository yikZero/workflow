Use message history when you need a complete audit trail showing every transformation a message underwent and in what order.

## Pattern

The workflow passes an envelope containing the payload and a history array through each step. Every step appends its action and outcome to the history before returning the updated envelope.

```typescript lineNumbers
type HistoryEntry = { step: string; action: string; timestamp: string; correlationId: string };
type Envelope = { payload: Record<string, unknown>; history: HistoryEntry[]; status: string };

declare function normalizeTicket(envelope: Envelope): Promise<Envelope>; // @setup
declare function classifySeverity(envelope: Envelope): Promise<Envelope>; // @setup
declare function chooseRoute(envelope: Envelope): Promise<Envelope>; // @setup
declare function dispatchTicket(envelope: Envelope): Promise<Envelope>; // @setup

export async function messageHistory(
  correlationId: string,
  subject: string,
  body: string,
) {
  "use workflow";

  let envelope: Envelope = {
    payload: { correlationId, subject, body },
    history: [],
    status: "processing",
  };

  envelope = await normalizeTicket(envelope);
  envelope = await classifySeverity(envelope);
  envelope = await chooseRoute(envelope);
  envelope = await dispatchTicket(envelope);

  return { ...envelope, status: "completed" };
}
```

## Key APIs

- [`"use workflow"`](/docs/api-reference/workflow/use-workflow) — declares the orchestrator function
- [`"use step"`](/docs/api-reference/workflow/use-step) — each processing stage is a durable step
