Use a wire tap when you need to inspect messages flowing through a pipeline for debugging or auditing without altering the main processing path.

## Pattern

Each step processes the message normally, then copies a snapshot to an audit trail array. The tap is non-invasive — it never mutates the message, only reads it.

```typescript lineNumbers
type OrderPayload = { orderId: string; item: string; quantity: number; [key: string]: unknown };

declare function validateOrder(message: OrderPayload, audit: unknown[]): Promise<OrderPayload>; // @setup
declare function enrichOrder(message: OrderPayload, audit: unknown[]): Promise<OrderPayload>; // @setup
declare function transformOrder(message: OrderPayload, audit: unknown[]): Promise<OrderPayload>; // @setup
declare function deliverOrder(message: OrderPayload, audit: unknown[]): Promise<OrderPayload>; // @setup

export async function wireTap(orderId: string, item: string, quantity: number) {
  "use workflow";

  const auditTrail: unknown[] = [];
  let message: OrderPayload = { orderId, item, quantity };

  message = await validateOrder(message, auditTrail);
  message = await enrichOrder(message, auditTrail);
  message = await transformOrder(message, auditTrail);
  message = await deliverOrder(message, auditTrail);

  return { orderId, status: "completed", auditTrail };
}
```

## Key APIs

- [`"use workflow"`](/docs/api-reference/workflow/use-workflow) — declares the orchestrator function
- [`"use step"`](/docs/api-reference/workflow/use-step) — each processing stage is a durable step
