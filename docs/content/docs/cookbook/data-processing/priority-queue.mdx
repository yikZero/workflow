Use priority queue when work items have different urgency levels and higher-priority items should be processed first.

## Pattern

Sort tasks by priority in a step, then process each sequentially in priority order. The sort step ensures the highest-priority items are always handled first.

```typescript lineNumbers
type Priority = "urgent" | "high" | "medium" | "low";
type TaskItem = { id: string; label: string; priority: Priority };

declare function sortTasks(tasks: TaskItem[]): Promise<TaskItem[]>; // @setup
declare function processTask(task: TaskItem, position: number): Promise<void>; // @setup

export async function priorityQueueFlow(tasks: TaskItem[]) {
  "use workflow";

  const sorted = await sortTasks(tasks);

  for (let i = 0; i < sorted.length; i++) {
    await processTask(sorted[i], i + 1);
  }

  return { processed: sorted.length };
}
```

## Key APIs

- [`"use workflow"`](/docs/api-reference/workflow/use-workflow) — declares the orchestrator function
